# PumpkinCrush

A mobile-friendly web game inspired by Underwatermelon: Fruit Merge, using placeholder photos. Designed for iPhone browser play.

## Features
- Merge-style gameplay
- Responsive design for mobile
- Placeholder images (replace with your girlfriend's photos)

## How to Play
- Drag and merge similar items to create new ones.
- Aim for the highest score by merging as many items as possible.

## Setup
- Open index.html in your mobile browser.

## Customization
- Replace placeholder images in the assets folder with your own photos.
